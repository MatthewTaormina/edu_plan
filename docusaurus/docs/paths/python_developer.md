# Python Developer Path

> **Who this is for:** You want to use Python professionally — scripting, APIs, automation, data tools.
> **Prerequisites:** None — start from the beginning, or jump in at your current level.
> **Time estimate:** 3–5 months at 1 hr/day

---

## The Sequence

```
Python Fundamentals   →   Data and Files   →   OOP   →   Professional Tools
    (6 lessons)             (4 lessons)       (5 lessons)     (4 lessons)
```

---

## Course 1 — Python Fundamentals
*~3 weeks · [Start →](../courses/python_fundamentals/index.md)*

Variables, control flow, loops, functions, and error handling. No prior experience needed.

**Done when:** You write a number guessing game that handles bad input gracefully.

---

## Course 2 — Python: Data and Files
*~2 weeks · [Start →](../courses/python_data_and_files/index.md)*

Lists, dicts, sets, file I/O, JSON, and CSV.

**Done when:** Your contact book CLI saves contacts to a JSON file and loads them on restart.

---

## Course 3 — Python: OOP
*~3 weeks · [Start →](../courses/python_oop/index.md)*

Classes, inheritance, dunder methods, and dataclasses.

**Done when:** Your task management system uses a proper class hierarchy with JSON persistence.

---

## Course 4 — Python: Professional Tools
*~2 weeks · [Start →](../courses/python_professional/index.md)*

Modules, virtual environments, CLI tools, logging, and testing with pytest.

**Done when:** You have a polished, tested CLI tool with a `pyproject.toml` and a clean test suite.

---
