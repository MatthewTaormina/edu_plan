# Next.js

> **Domain:** Web Development · **Time:** 10–12 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [React Fundamentals](../react_fundamentals/index.md)

> **Who needs this:** React developers building production applications. Next.js is the most popular React meta-framework and the recommended way to start a React project per the official React docs.

---

## 🎯 Course Objectives

- [ ] Create a Next.js 14+ App Router project
- [ ] Understand Server Components vs Client Components
- [ ] Build file-based routes with layouts, loading, and error files
- [ ] Fetch data in Server Components (no useEffect)
- [ ] Implement SSG and ISR for static content
- [ ] Build API routes with Route Handlers
- [ ] Add authentication patterns and middleware
- [ ] Deploy to Vercel
