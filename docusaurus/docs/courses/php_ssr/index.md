# PHP & Server-Side Rendering

> **Domain:** Web Development — Backend · **Time:** 8–10 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [HTML Fundamentals](../html_fundamentals/index.md) · [CSS Fundamentals](../css_fundamentals/index.md)

> **Tool:** PHP 8.x · **Introduced:** 1996 · **Latest:** 8.3 (2024) · **Deprecated:** N/A · **Status:** 🟢 Modern

> **Who needs this:** Developers working with WordPress, Drupal, cPanel/shared hosting environments, or any PHP-based codebase. PHP still powers ~77% of all websites with a known server-side language (WordPress alone accounts for 43%).

---

## 🎯 Course Objectives

- [ ] Install PHP and run the built-in development server
- [ ] Understand PHP's request lifecycle and how it differs from Node.js
- [ ] Serve dynamic HTML by embedding PHP in templates
- [ ] Read and validate form submissions (`$_GET`, `$_POST`)
- [ ] Manage user state with sessions and cookies
- [ ] Protect forms against CSRF attacks
- [ ] Build a flat-file blog without a database
