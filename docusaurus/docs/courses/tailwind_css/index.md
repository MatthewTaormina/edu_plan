# Tailwind CSS

> **Domain:** Web Development · **Time:** 6–8 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [CSS Fundamentals](../css_fundamentals/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Who needs this:** Anyone building UI with React, Angular, or any component-based framework. Tailwind is the dominant CSS strategy in modern frontend teams.

:::note
Tailwind CSS requires a build tool (Vite, webpack, etc.) to scan your files and purge unused styles. Complete [Build Tools & Toolchain](../build_tools/index.md) first.
:::

---

## 🎯 Course Objectives

- [ ] Explain the utility-first CSS philosophy
- [ ] Apply Tailwind utilities for spacing, typography, color, and layout
- [ ] Build responsive layouts using Tailwind's breakpoint system
- [ ] Implement dark mode with Tailwind's `dark:` variant
- [ ] Create a custom design system via `tailwind.config.js`
- [ ] Extract reusable styles using `@apply` and component patterns
- [ ] Integrate Tailwind with a Vite project
- [ ] Complete a capstone: Admin Dashboard UI
