# Python: Professional Tools

**Level:** Intermediate · **Prerequisites:** [Python Fundamentals](../python_fundamentals/index.md) · **Lessons:** 4 · **Duration:** ~5 hours

> By the end of this course you will manage Python projects with virtual environments,
> build polished CLI tools, write structured logs, and run your project in CI.

---

## Lessons

| # | Lesson | Time |
|---|--------|------|
| [01](lesson_01.md) | Modules, packages, and virtual environments | 75 min |
| [02](lesson_02.md) | Building CLI tools — argparse and logging | 75 min |
| [03](lesson_03.md) | Testing with pytest | 60 min |
| [04](lesson_04.md) | **Capstone** — polished CLI tool with tests | 90 min |

---

## What You'll Build

A **polished CLI tool** that:
- Accepts structured arguments with help text
- Logs activity to a file and terminal
- Has a test suite with pytest
- Uses a proper project structure with `pyproject.toml`
