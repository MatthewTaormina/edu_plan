# Express API

> **Domain:** Web Development — Backend · **Time:** 10–12 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [Node.js Fundamentals](../nodejs_fundamentals/index.md)

> **Who needs this:** Developers building REST APIs with Node.js. Express is the most widely deployed Node.js web framework and is used by companies of all sizes.

---

## 🎯 Course Objectives

- [ ] Set up an Express application with TypeScript
- [ ] Define modular routes and routers
- [ ] Write, compose, and order middleware correctly
- [ ] Parse request bodies, params, and query strings
- [ ] Handle errors consistently across the API
- [ ] Validate input with Zod
- [ ] Implement JWT and session-based authentication
- [ ] Upload files with Multer
- [ ] Test API endpoints with Vitest + Supertest
- [ ] Build and ship a fully functional REST API

---
