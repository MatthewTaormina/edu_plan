# React Fundamentals

> **Domain:** Web Development · **Time:** 10–12 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [TypeScript Fundamentals](../typescript_fundamentals/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Who needs this:** Anyone building modern web UIs. React is the most-used JavaScript UI library in the world, with the largest ecosystem and highest job-market demand.

---

## 🎯 Course Objectives

- [ ] Create a Vite + React + TypeScript project from scratch
- [ ] Write functional components using JSX/TSX
- [ ] Pass data between components using props and children
- [ ] Manage local component state with `useState`
- [ ] Synchronise with external systems using `useEffect`
- [ ] Handle forms, events, and controlled inputs
- [ ] Render lists efficiently with `key` and compose conditional UI
- [ ] Share state across a subtree using `Context` and `useContext`
- [ ] Abstract reusable logic into custom hooks
- [ ] Complete a capstone: Reading List Tracker
