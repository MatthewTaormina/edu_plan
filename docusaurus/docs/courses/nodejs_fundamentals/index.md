# Node.js Fundamentals

> **Domain:** Web Development — Backend · **Time:** 8–10 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [Vanilla JavaScript Fundamentals](../javascript_fundamentals/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Who needs this:** Anyone building server-side JavaScript applications. This course is the entry point for the Express API, Next.js, and any Node.js-based backend course.

---

## 🎯 Course Objectives

- [ ] Explain how the Node.js event loop enables non-blocking I/O
- [ ] Use CommonJS and ES Module systems
- [ ] Work with the file system, paths, and streams
- [ ] Build a raw HTTP server with the `http` module
- [ ] Manage environment variables and process signals
- [ ] Write async code with callbacks, Promises, and async/await
- [ ] Package and publish a simple CLI tool

---

## 📖 Lessons

| # | Lesson | Time |
|---|--------|------|
| 01 | What is Node.js — Runtime, Event Loop, Non-Blocking I/O | 60 min |
| 02 | Modules — CommonJS, ESM, and npm | 45 min |
| 03 | File System & Streams | 60 min |
| 04 | The HTTP Module — Raw Servers | 45 min |
| 05 | Environment, Process & CLI Args | 45 min |
| 06 | Async Patterns — Callbacks → Promises → async/await | 60 min |
| 07 | Capstone: CLI File Processor | 90 min |
