# Python: Data and Files

**Level:** Beginner · **Prerequisites:** [Python Fundamentals](../python_fundamentals/index.md) · **Lessons:** 4 · **Duration:** ~5 hours

> By the end of this course you will know how to organise data using Python's built-in
> collections, read and write files, handle JSON and CSV, and build a file-backed
> data store.

---

## Lessons

| # | Lesson | Time |

| [03](lesson_03.md) | JSON and CSV | 45 min |
| [04](lesson_04.md) | **Capstone** — JSON-backed contact book | 60 min |

## What You'll Build

A **contact book CLI** that stores contacts in a JSON file.
Supports add, search, list, and delete.
