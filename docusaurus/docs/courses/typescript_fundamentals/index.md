# TypeScript Fundamentals

> **Domain:** Web Development · **Time:** 6–8 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [Vanilla JavaScript Fundamentals](../javascript_fundamentals/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Who needs this:** Anyone heading to React or Angular. Both ecosystems expect TypeScript. It also makes pure JavaScript projects dramatically more maintainable.

---

## 🎯 Course Objectives

- [ ] Set up a TypeScript project with `tsconfig.json` and strict mode
- [ ] Apply types to variables, functions, objects, and arrays
- [ ] Create reusable types with interfaces and type aliases
- [ ] Write generic functions and generic types
- [ ] Apply built-in utility types to transform existing types
- [ ] Type DOM events and elements without type errors
- [ ] Complete a capstone: a fully typed, browser-based Todo application
