# Build Tools & Toolchain

> **Domain:** Web Development · **Time:** 4–5 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [Vanilla JavaScript Fundamentals](../javascript_fundamentals/index.md)

> **Who needs this:** Anyone starting a framework-based project. Understanding your toolchain removes the magic from `npm run dev` and prevents lost hours debugging configuration issues.

---

## 🎯 Course Objectives

- [ ] Explain what Node.js is and why it's needed for frontend tooling
- [ ] Manage packages with npm and pnpm
- [ ] Understand ES Modules, bundling, and tree-shaking
- [ ] Configure and use Vite as a dev server and bundler
- [ ] Set up ESLint and Prettier for code quality
- [ ] Configure Vitest for unit testing
