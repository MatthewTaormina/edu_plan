# Laravel Fundamentals

> **Domain:** Web Development — Backend · **Time:** 10–12 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [PHP & Server-Side Rendering](../php_ssr/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Tool:** Laravel 11 · **Introduced:** 2011 · **Latest:** 11.x (2024) · **Deprecated:** N/A · **Status:** 🟢 Modern

> **Who needs this:** PHP developers building production applications. Laravel is the dominant PHP MVC framework used in professional web development, SaaS products, and enterprise applications.

---

## 🎯 Course Objectives

- [ ] Scaffold a Laravel project and understand the MVC structure
- [ ] Define routes and map them to controller methods
- [ ] Build templates with the Blade templating engine
- [ ] Understand the request/response lifecycle and middleware
- [ ] Use Eloquent models to represent data (without raw SQL setup)
- [ ] Add authentication with Laravel Breeze
- [ ] Build a complete blog with auth-protected admin routes
