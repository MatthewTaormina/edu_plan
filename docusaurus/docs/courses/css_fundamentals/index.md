# CSS Fundamentals

> **Domain:** Web Development · **Time:** 8–10 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [HTML Fundamentals](../html_fundamentals/index.md)

> **Who needs this:** Anyone who wants to make web pages look good. After HTML teaches you *what*, CSS teaches you *how it looks*.

---

## 🎯 Course Objectives

- [ ] Understand how CSS is linked to HTML and how rules are applied
- [ ] Master the cascade, inheritance, and specificity
- [ ] Control layout with Flexbox and CSS Grid
- [ ] Apply responsive design using media queries and relative units
- [ ] Add polish with animations and transitions
- [ ] Complete a capstone: a responsive, animated profile landing page

---

## 📖 Lessons

| # | Lesson | What you'll build |
