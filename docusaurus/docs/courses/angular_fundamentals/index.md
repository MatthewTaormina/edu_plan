# Angular Fundamentals

> **Domain:** Web Development · **Time:** 12–14 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [TypeScript Fundamentals](../typescript_fundamentals/index.md) · [Build Tools & Toolchain](../build_tools/index.md)

> **Who needs this:** Developers building enterprise-grade frontends or working in organisations that standardise on Angular. Angular is the dominant choice in large government, financial, and healthcare organisations.

---

## 🎯 Course Objectives

- [ ] Create an Angular 17+ project with the Angular CLI
- [ ] Write standalone components with signals-based reactivity
- [ ] Pass data with `@Input()` / `@Output()` and the new input function
- [ ] Use the Angular Router for multi-page navigation
- [ ] Call REST APIs with `HttpClient`
- [ ] Build template-driven and reactive forms with validation
- [ ] Use Angular pipes and directives
- [ ] Write unit tests with Jasmine + Angular Testing Utilities
