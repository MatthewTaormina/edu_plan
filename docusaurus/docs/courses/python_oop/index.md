# Python: Object-Oriented Programming

**Level:** Intermediate · **Prerequisites:** [Python Fundamentals](../python_fundamentals/index.md) · **Lessons:** 5 · **Duration:** ~6 hours

> By the end of this course you will be able to design programs using classes,
> understand inheritance and composition, and write idiomatic Python OOP including
> dunder methods and dataclasses.

---

---

## What You'll Build

A **task management system** using classes:
- `Task` dataclass with title, due date, priority, status  
- `TaskList` class with add, complete, filter, sort
- Persistent storage via JSON (from the Data and Files course)
