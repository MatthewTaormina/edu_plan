# 🧭 Orientation — How to Use This Repo

> Read this first. It takes ~15 minutes and will save you hours of confusion.

---

## What This Is (and Isn't)

✅ **This IS:**
- A structured navigation system for self-learning IT, programming, and DevOps
- A collection of curated resources, research questions, and hands-on projects
- Designed for self-directed learners — especially those with ADHD or autism

❌ **This IS NOT:**
- A replacement for practice — you must write code to learn
- A complete course — it points you to resources, then you do the work
- Linear — you can start at many different points

---

## How the Repo Is Organized

```
You → Pick a Path → Follow domain units → Build projects → Repeat
```

1. **Paths** (`/paths/`) define a recommended sequence of domains for a specific goal (web dev, systems, etc.)
2. **Domains** (`/domains/`) contain atomic units — focused, completable chunks
3. **Projects** (`/projects/`) combine multiple concepts into real things you can show people
4. **Resources** (`/resources/`) is a curated list of courses and reference material

---

## ⏱️ Time Expectations

| Commitment | What You Can Do |
|------------|----------------|
| 30 min/day | ~1 unit/week, slow but steady |
| 1–2 hrs/day | ~1 unit/3 days — a solid pace |
| Full-time | 2–4 units/week — aggressive pace |

**Realistic full coverage estimate:** 18–36 months at part-time pace.
**Foundations → First job ready:** ~6–12 months, focused on one path.

---

## 🧠 ADHD Study Tips Built Into This System

1. **Time-box sessions** — Use a 25-minute Pomodoro timer. This repo's units are sized for it.
2. **One unit at a time** — Don't open 10 tabs. Finish the current unit's checklist first.
3. **Mark things done** — Check off `[ ]` boxes. Your brain needs to *see* progress.
4. **Use the kanban** — Move cards. Visual progress matters.
5. **Allow exploration** — Research questions are marked 🔍 and are *optional* rabbit holes.
6. **Earn badges** — When you finish a milestone, log it in `.meta/kanban/board.md`.

- [ ] GitLens
- [ ] Error Lens

- [ ] **Obsidian** — For personal notes alongside this repo
