# React Advanced

> **Domain:** Web Development · **Time:** 8–10 hours total · **Platform:** Windows + Linux
> **Prerequisites:** [React Fundamentals](../react_fundamentals/index.md)

> **Who needs this:** React developers who want to write production-grade code: optimised, well-tested, and with proper routing and server data management.

---

## 🎯 Course Objectives

- [ ] Prevent unnecessary re-renders with `useMemo`, `useCallback`, and `memo`
- [ ] Model complex, multi-step state with `useReducer`
- [ ] Build multi-page SPAs using React Router v6
- [ ] Manage server state with TanStack Query (caching, loading, errors, refetching)
- [ ] Use Error Boundaries and Suspense for graceful failure and loading states
- [ ] Write component and hook tests with Vitest + Testing Library
