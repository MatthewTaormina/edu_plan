# HTML Fundamentals

> **Domain:** Web Development · **Time:** 6–8 hours total · **Platform:** Windows + Linux
> **Prerequisites:** None — this is your first step into web development.

> **Who needs this:** Anyone who wants to build websites or understands how the web works. Before you can style or script a page, you must be able to structure it.

---

## 🎯 Course Objectives

- [ ] Understand what HTML is and how browsers parse it
- [ ] Build a complete, valid HTML document from scratch
- [ ] Use headings, paragraphs, links, images, lists, and tables correctly
- [ ] Build accessible forms that collect user data
- [ ] Write semantic HTML that screen readers and search engines understand
- [ ] Complete a capstone project: an accessible, multi-section recipe page

---

## 📖 Lessons

| # | Lesson | What you'll build |
|---|--------|-------------------|
| 01 | The Web Document | Your first `index.html` |
| 02 | Text & Flow | A formatted article |
| 03 | Anchors & Navigation | A linked page with a nav bar |
| 04 | Immersive Media | An image gallery page |
| 05 | Table Data | A schedule table |
| 06 | Forms & Input | A registration form |
