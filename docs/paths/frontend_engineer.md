# Frontend Engineer Path

> **Goal:** Build production-ready, accessible, performant frontend applications
> **Time Estimate:** 6–8 months (10–15 hrs/week)
> **Entry Level:** Some HTML/CSS experience helpful but not required

---

## 📍 What This Path Covers

The Frontend Engineer path takes you from HTML fundamentals to building modern, production-grade web applications with React, Next.js, or Angular. You will be comfortable with the full modern frontend stack:

- HTML, CSS, Vanilla JavaScript
- TypeScript (strict mode)
- React (hooks, context, routing, server-state, testing)
- Next.js (SSR, SSG, ISR, API routes)
- Angular (standalone components, signals, reactive forms)
- Tailwind CSS
- Build tooling (Vite, ESLint, Vitest)
- Performance and accessibility fundamentals

At the end of this path you'll be ready for **Frontend Engineer**, **React Developer**, or **Angular Developer** roles.

## 🏁 You're Ready When You Can…

- [ ] Build a fully responsive, accessible webpage from scratch
- [ ] Set up a Vite + TypeScript + React (or Angular) project from the CLI
- [ ] Fetch data from a public API and display it with loading/error states
- [ ] Add client-side routing with multiple pages
- [ ] Manage global app state (Auth, Theme) with Context (React) or Services (Angular)
- [ ] Write at least 10 passing tests
- [ ] Deploy a production build to Vercel, Netlify, or GitHub Pages
